# VideoCreatePlateform
这是本人的毕业设计 视频创作平台；

APP录制视频上传到后台，从后台拉取数据播放视频或轮播图；
录制预览编码用AudioRecord，Camera2，ffmepg和OpenGLES；
解码也是ffmpeg和OpenGLES；
音乐解码和播放是ffmpeg和AudioTrack；

作者: 李凤强
